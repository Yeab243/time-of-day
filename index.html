This page demonstrates how to use a JavaScript to Date object to integrate information
from the client's system clock into a web page. The web page uses JavaScript to determine
the current hour, and presents a different image, custom message, and color based on the
specific time of day. The page also produces a dynamic copyright, produced by JavaScript.
